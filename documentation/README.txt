---Installing Dependencies---
In terminal: pip3 install numpy==1.23.5
In terminal: pip3 install pandas==1.5.2
In terminal: pip3 install -r documentation/requirements.txt
In terminal: pip3 install matplotlib==3.5.3
In terminal: python3 age_verification.py

---Usage---
python3 age_verification.py

---Output---
Graph and Data file will be saved in output folder
data.csv contains the output data. The data will be organized from least occupied to most occupied time per venue. With the corresponding average occcupany rate listed in the count colum.
graph.png will visualize the data located in data.csv

**Sample Outputs have already been generated and can be found in the output folder**
